# Cybrmicro
Color flavor for Micro inspired by the color palette popularized by **Cyberpunk 2077**.

## Result
<img src="../assets/inspiration/insp-micro.png" width="800"/></td>

## What to do
### Create theme file:

```sh
micro ./.config/micro/colorschemes/cybrmicro.micro
```

### Insert [cybrmicro.micro](../micro/cybrmicro.micro)

### Set theme:
```sh
micro
# press CTRL+E

set colorscheme cybrmicro.micro
```
